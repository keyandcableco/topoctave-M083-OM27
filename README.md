# topoctave-M083-OM27

Building on this amazing work to adapt for Omnichord OM-27's M083 tone generator.

M082/A-M083/A-M086/A
TONE GENERATORS

The M082/A, M083/A and M086/A are monolithic tone generators specially designed for electronic organs. 
The only difference between the
M082, M083, M086 and the M082A, M083А,
M086A is the maximum input clock frequency,
which is 4500KHz for the standard types and
2500KHz for the "A" types. Constructed on a
single chip using low threshold N-channel silicon
gate technology they are supplied in a 16 lead
dual in-line plastic package.
